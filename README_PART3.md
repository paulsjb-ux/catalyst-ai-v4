# Catalyst AI v4 — Sprint 1 Part 3

Adds:
- Live Yahoo Finance market-data download
- Starter universe and custom ticker input
- SMA20, SMA50, SMA200, RSI14 and relative-volume indicators
- 0–100 candidate scoring
- BUY, WATCH and IGNORE signals
- Live dashboard metrics
- CSV scan export

## Install

Replace the supplied files in the existing repository, then run:

```bash
source .venv/bin/activate
pip install -r requirements.txt
python -m streamlit run app.py
```

Start with 20–30 tickers while testing.
